## virt-helpers/

### ubuntu-16.04-builder.sh

provision an Ubuntu 16.04 machine using virt-builder.

Example: `ubuntu-16.04-builder.sh app-test`
Results in the file `app-test.qcow2` which is a bare metal Ubuntu 16.04 installation.
